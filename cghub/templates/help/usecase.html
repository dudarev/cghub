Help: Use Case

Use Case Example.


  1. Navigate to: https://browser.cghub.ucsc.edu/
  2. Pick the filters that you wish to use for your query by utilizing the left side filter bar.
  3. Apply your filters from the left side filter bar by clicking on "Aplly filters".
  4. Select the BAM files you wish to download, and add them to your cart. Remember you can continue to peruse the results by clicking the "next" page button and/or changing the number of results displayed per page {10, 25, 50}.
  5. Once you have the files you are looking for added to your cart, click on the cart in the top navigation bar (which should have the corresponding number of files you selected to add to your cart. E.g. "Cart (72)")
  6. Once you are on the "Cart" page you can do a final review of your files that will be added to your XML file.
  7. Once you have verified the files in your cart, click Download Manifest to create a XML file that you can pass to GeneTorrent to download, or click Download metadata to create a XML file that you can examine.
  8. In this use case I will create a "Download Manifest" XML file that I can then pass to me GeneTorrent client. (please note that GeneTorrent is not yet available for Windows® or Mac® OS's) Be sure to note where your Manifest Download file is download and the name of the XML file.
  9. GeneTorrent (GT) command:

    GeneTorrent -d /path/to/manifest.xml -c /path/to/mykey.pem -p /path/to/save/data -vv

    • The -d option tells GT to look at this file to determine what to download.
    • The -c option tells GT to look at this location for the mykey.pem file.
    • The -p opteion tells GT what path to download the BAM files to.
    • And the -vv (two v's) is a verbose output to the screen so you can see what GT is doing.

    All these options can be viewed from the manual (man) pages from GT in the command line ( i.e. man GeneTorrent) or downloaded from CGHub's website found here under "CGHub Documentation":
    https://cghub.ucsc.edu/help/help.html
