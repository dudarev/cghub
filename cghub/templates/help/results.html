Help: Overview

Results Area.

The results area has seven areas of interest.

  1. You will see a total number of results found under the Found ### results.
  2. The Applied filter(s): will list the filters used in this search.
  3. Select all: Select all will place a check mark in each box displayed on the page.
  4. Add to cart: Add to cart will add any selected items to your cart (the cart, and xml files that can be created will be cover in the next section).
  5. 10, 25, 50: You also have the option to list multiple items per page, Ie 10, 25, or 50 items per page (In order to select more items at once, if you have a large number of items to download).
  6. The main results area will display the items the search produced (up to 10, 25, or 50 depending on which option you selected in section 3.1.5)

    Available fields that can be displayed in the main area are as followed (by default all column headers are selected).

    - Barcode
    - Analysis Id
    - Accession
    - Files Size
    - Last Modified
    - Disease
    - Sample Type
    - Experiment Type
    - Library Type
    - Center

    Each of these column headers can be deselected at any time if you do not wish to view them. Simply hold your pointer over any column header and a pop-up list with the column headers and check boxes will appear.

    The column size can also be adjusted by placing your pointer on the line that separates the columns. Once your pointer is on the column divider line, a double arrow will appear, which you can use to adjust the width of the columns
  7. Prev, 1 2 3...Next: Once you have selected the items on the current page and added them to your cart, you can page through the rest of the results by clicking the prev or next button, or the page number at the bottom of the main results area i.e. 1, 2, 3...
