Batch search

The Batch search facility takes a list of identifiers and searches CGHub for all matching datasets.

The following types of identifiers may be used:

  • Sample barcode - This is a terse, human-readable identifier See the TCGA Wiki for a descriptions the TCGA barcode fields: https://wiki.nci.nih.gov/display/TCGA/TCGA+Barcode
  • analysis id - UUID of dataset at CGHub. This is a stable id that uniquely identifies one primary data file, that is a BAM or a FASTQ.
  • aliquot id - UUID of the aliquot that was sequenced. A single aliquot maybe have multiple CGHub multiple datasets.
  • sample id - UUID of the sample that was sequenced. A single sample maybe have multiple CGHub multiple datasets.
  • participant id - UUID of the individual from which the sample was obtained. A given patient maybe have multiple CGHub multiple datasets.

Batch search identifiers may be entered by two methods:

  1. In the text input box, one per line
  2. Uploading a text file that contains one identifier per line

After submitting, the found datasets and unknown ids are displayed. The list of found datasets may have entries removed and then the remainder added to the cart.
