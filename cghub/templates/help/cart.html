Help: Cart

Cart Page.


The cart page will look very similar to the main search/results page, but it will have three new action buttons.

  1. Remove from cart.

    This will allow you to remove any items you decide you no longer want.

  2. Download Manifest

    This will download a Manifest.xml file that you can pass to GeneTorrent to download the actual BAM files from CGHub. Example GeneTorrent command line: GeneTorrent -d /path/to/manifest.xml -c /path/to/mykey.pem -p /path/to/save/data â€“vv

  3. Download Metadata

    This will download a metadata file, which you can examine, and possibly extrapolate additional information to determine if you have the specific BAM files you need for your experiments or studies.

    Note: XML files can be hard for humans to read in a standard text editor. You may need additional software to determine what information is relevant in the xml file if you decide to decipher its contents.
