Accessibility

CGHub is committed to making the data available from CGHub accessible to all users. If you are using special tools to access the internet, and encounter problems with CGHub websites, or tools, please email CGHub Support <support@cghub.ucsc.edu or 831-459-5243>, and an alternative method will be provided to query the CGHub database. Please note that in addition to the web based data browser tool, CGHub offers a command line tool called cgquery bundled with the GeneTorrent program (located here: https://cghub.ucsc.edu/software/downloads.html). In addition, a TSV (tab-separated value) file called "Data Manifest File in TSV format" is available for download from: https://cghub.ucsc.edu/summary_stats.html

This "Data Manifest File in TSV format" file (updated every 24 hours) can be used to identify important information related to files users may wish to download. Please contact CGHub Support <support@cghub.ucsc.edu or 831-459-5243> if you would like file sent to you which contains all of the metadata associated with any analysis_id's (UUIDs) in question.
