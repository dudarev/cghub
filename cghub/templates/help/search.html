Help: Search

Search for any query by entering it into the search box. For example, TCGA-50-5931.

With some attributes partial wildcards work. For instance, 79ad1a*. This does not work for barcodes.

To see more content in any column adjust its width by dragging lines between columns.

Results may be filtered and sorted. To filter open the category for filtering in the sidebar and click the attribute. If filters is used one may see a message what filters are used. It is possible to use several filters.

To sort by values in a column click on its header. To change the sorting from ascending to descending click the header again. It is only possible to sort by values in one column.
