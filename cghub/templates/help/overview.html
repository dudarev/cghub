Help: Overview

The CGHub Data Browser enables users to explore the cancer genomics data at CGHub. The application use a familiar shopping cart paradigm. The user browses the available data sets using filters and text searches. Details of individual data sets can be examined by clicking on the entries in the resulting list. Some or all of the resulting data sets are then added to the cart.

From the cart, the user may create a manifest file to pass to the gtdownload high-performance transfer client to download the data sets. The user may also download a summary of the results in TSV (tab-separated values) format, suitable for import into a spreadsheet or download the complete metadata in SRA XML format.

The CGHub Data Browser makes extensive use of tooltips to assist in using the browser. Most user interface elements and menu items will pop-up a description if the mouse hovers over the item for 1/4 of a second. Many of the values in the data list, particularly acronyms and abbreviations, also have tooltips to describe their meaning.

Filters

The primary method of searching for subsets of data is by specifying filters in the left panel. Both the meaning of the filters and their values are documented by tooltips displayed when hovering over menus. The results will be the data sets matching the intersection of the filters and a text search, as described below.

Text Search

A full-text search of the metadata is also supported by entering text in the Search box that appears on the right side of the menu bar. For example, a bar code ID can be entered in this box to search for its occurrence in the metadata.

Text search results are restricted by the currently applied filters.

To search the metadata for a phrase (more than one word separated by spaces) enclose the item in quotations (for example: "Illumina TruSeq").

Multiple text search terms can be entered in the Search box by separating them with spaces. Results are provided for only entries that meet all the specified search terms. For example, searching for "RNA-Seq Illlumina" will select only entries having references to both "RNA-Seq" and "Illumina" in their metadata.

Note: English text in the metadata is very limited. Searches for terms like disease name will produce limited or no results. Use the filters whenever possible.
